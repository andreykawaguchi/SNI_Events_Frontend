💡 COMO USAR EM 3 LINHAS
═══════════════════════════════════════════════════════════════════

  1. Criar schema:
     export const mySchema = z.object({ email: z.string().email() });

  2. Usar em componente:
     const { register, handleSubmit, formState: { errors } } = useForm({
       resolver: zodResolver(mySchema),
     });

  3. Render:
     <input {...register('email')} />
